⚙️ Configura Firebase

Para que el inicio de sesión funcione necesitas crear un proyecto gratuito en Firebase y pegar tu configuración aquí.

1. Ve a console.firebase.google.com
2. Crea un proyecto → activa Authentication
3. Habilita Email/Password y Google en Sign-in methods
4. Crea una base de datos Firestore
5. Ve a Configuración del proyecto → Tu app → copia el objeto firebaseConfig
